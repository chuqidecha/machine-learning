## 类别特征分析
1. 创建一张表存储各个类别特征的出现次数。
```sql
CREATE TABLE train_stat(              
    ID INTEGER PRIMARY KEY AUTOINCREMENT, 
    COL_NAME CHAR(32),                    
    COL_VALUE CHAR(32),                   
    VALUE_COUNT INTEGER                   
);                                    
```
